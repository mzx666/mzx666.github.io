如何搭建一个博客呢？

发表于2021-02-22|更新于2021-04-16

教大家搭建一个博客（小白适用）！

目录

  • 准备什么？
  • 注册 Github

准备什么？

  • 一台能上网的电脑
  • 一个电子邮箱（可以接收邮件）
  • 我的博客，没错，你已经打开了~
  • 支付宝（准备大概十块钱）（选用）
  • 检查自己的作业已全部完成，并且上交了！！！！

Ps：一定要准备好以上东西（选用的选用）

注册 Github

接下来我们需要先注册一个Github账号,因为它可以为我们提供一个静态网站服务。（看不懂没关系）

首先登陆github主页：URL:https://github.com/ 传送门

然后单击Sign up(在右上角)

紧接着填完整个人信息，过人机验证。

随后，github会往你的邮箱发送一个验证邮件，验证即可~
再返回到主页进行登陆~

登陆完成后，接着看文章~
